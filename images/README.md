# Images

Drop these 4 files into this folder and they'll appear on the site automatically.
The base name must match exactly (case-sensitive). The extension can be
**`.jpg`, `.jpeg`, `.png`, or `.webp`** — the page tries them in that order and uses
whichever one it finds.

| Base name       | Where it shows | Suggested crop / size          |
|-----------------|----------------|--------------------------------|
| `hero-sheila`   | Hero (top)     | Portrait 4:5, ~1000×1250px     |
| `production`    | About the show | Portrait 4:5, ~1000×1250px     |
| `jay-hill`      | Artists        | Square 1:1, ~440×440px         |
| `adam-khomsi`   | Artists        | Square 1:1, ~440×440px         |
| `HIO-logo`      | Artists        | Square-ish, transparent PNG ok |

So `production.jpg`, `production.jpeg`, `production.png`, and `production.webp` are all
fine — use whichever you have. (WhatsApp saves as `.jpeg`, which now works too.)

Tips:
- `.jpg` is best for photos, `.png` for anything with sharp edges/text or transparency.
- Export photos at ~80% quality — no need for anything huge.
- Until a matching file exists, that spot shows a broken-image icon.
